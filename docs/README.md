# SeqFlash documentation

This folder holds longer-form design documentation and milestone acceptance notes.

- `architecture/` — architecture overviews and diagrams
- `adr/` — Architecture Decision Records (one file per decision)
- `formats/` — FASTA / FASTQ format notes and edge cases
- `performance/` — [methodology](./performance/methodology.md), [results](./performance/results/) (templates + Criterion archives)
- `milestones/` — acceptance checklists ([M7](./milestones/M7_ACCEPTANCE.md), [M8](./milestones/M8_ACCEPTANCE.md))

The authoritative specification is [`../SeqFlash_DEVELOPMENT_PLAN.md`](../SeqFlash_DEVELOPMENT_PLAN.md).
